# КБЖУ Калькулятор

Простое Android-приложение для расчёта калорийности и процентного соотношения белков, жиров, углеводов (КБЖУ).

## Функционал
- **Экран ввода**: ввод значений калорий, белков, жиров, углеводов
- **Экран расчёта**: просмотр введённых данных и запуск расчёта
- **Экран результата**: отображение общей калорийности и процентного соотношения БЖУ

## Технологии
- Kotlin
- Jetpack Compose
- Navigation Compose
- Material Design 3

## Сборка

### Вариант 1: Сборка через GitHub Actions (рекомендуется)
1. Загрузите этот проект в свой репозиторий на GitHub.
2. Настройте Secrets (не требуется для сборки debug APK).
3. При пуше в ветку `main` автоматически запустится сборка.
4. Готовый APK будет доступен в разделе Actions → последний workflow → Artifacts, либо в Release.

### Вариант 2: Локальная сборка
```bash
./gradlew assembleDebug
```
APK будет создан в `app/build/outputs/apk/debug/app-debug.apk`

### Вариант 3: Открыть в Android Studio
1. Импортируйте проект.
2. Нажмите "Run" (зелёная стрелка) для запуска на эмуляторе или реальном устройстве.

## Проверка через онлайн-сервисы
- **Appetize.io**: загрузите APK, получите ссылку на эмулятор в браузере.
- **BrowserStack**: также поддерживает загрузку APK для тестирования.
- **GitHub Pages + Android Online Emulator**: можно использовать сервисы вроде "Android JS".
